# supermarket_release_v1

Production-style asset pack for the supermarket game.

## Usage
1. Extract this zip into your project root.
2. Keep the `public/assets/...` paths unchanged.
3. In Phaser, load assets with `/assets/...` web paths.

## Included
- Common UI icons and panels
- Storefront / outer-store assets
- Day 1 gameplay assets
- Day 1 delivery module
- Day 2 opening stock assets
- Day 2 promotion wing assets
- Day 3 folder scaffolding for later uploads
