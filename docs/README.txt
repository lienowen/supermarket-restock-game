Supermarket Restock Game - usable art pack

This package contains a recommended set of replacement / expansion images,
plus earlier images moved into legacy_not_recommended for comparison.

Recommended files (direct use)
1) public/assets/storefront/storefront_day.png
   New supermarket lobby/day background.

2) public/assets/day01/delivery/boxes/box_milk.png
   Milk delivery box replacement.

3) public/assets/storefront/modules/promo_endcap.png
   Promotional endcap module.

4) public/assets/storefront/modules/shelf_snacks_pantry.png
   Snacks / pantry shelf module.

5) public/assets/storefront/modules/fridge_dairy.png
   Dairy fridge module.

6) public/assets/storefront/modules/checkout_counter.png
   Checkout / service counter module.

7) public/assets/storefront/modules/produce_island.png
   Produce display module.

8) public/assets/storefront/modules/shelf_drinks.png
   Drinks shelf module.

Legacy folder
- legacy_not_recommended/
  Earlier generated images that were considered less suitable or inconsistent.

Suggested usage
- Replace the lobby with storefront_day.png.
- Replace the milk box with box_milk.png.
- Use the modules folder for future expansion and composition.
