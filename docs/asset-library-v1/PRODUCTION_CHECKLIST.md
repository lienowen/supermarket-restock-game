# 素材生产与验收清单

## 单文件验收

- [ ] Key、路径、文件名与素材清单完全一致
- [ ] 无文字烘焙；场景标牌必须有多语言替换方案
- [ ] 透明边缘干净，无蓝底、白边、黑边和残留像素
- [ ] 画布、比例和锚点符合所属类别标准
- [ ] 固定第三人称透视与左上光向一致
- [ ] 阴影方向、软硬度和透明度与同类素材一致
- [ ] 无真实商标、品牌和不明版权来源
- [ ] 压缩后无明显色带、噪点和锯齿

## 角色动作组

- [ ] idle、walk、carry、open、restock、scan 的脚底锚点一致
- [ ] 身高和身体比例不随动作漂移
- [ ] 手持道具位置与独立道具锚点匹配
- [ ] 朝向、服装、肤色和光照连续

## 货架与商品

- [ ] 空、少、半满、较满、满、错放状态外轮廓一致
- [ ] 商品不穿透层板、门框或相邻商品
- [ ] 错放商品清晰可识别但不过度夸张
- [ ] 单品、整箱、货架簇使用同一 SKU 视觉语言

## UI

- [ ] 文本与背景分离，支持多语言和动态数字
- [ ] 触控区域不低于 44 逻辑像素
- [ ] 普通、按下、禁用、选中状态齐全
- [ ] 750×1334、桌面横屏和常见手机比例下不遮挡玩法区域

## 音频

- [ ] OGG 可用，必要时提供 MP3 兼容格式
- [ ] 短音效没有长尾静音
- [ ] 同组随机变体响度一致
- [ ] BGM 和环境循环无爆点、断点

## 发布检查

- [ ] 概念图、PSD、BLEND、AEP 和参考板未进入 dist
- [ ] 首关只预加载必需素材
- [ ] 未引用的大型素材被发布裁剪脚本移除
- [ ] 低端手机测试无明显纹理内存峰值
- [ ] 慢网或加载失败时有占位、重试和错误提示
- [ ] 所有 production 素材均在 Asset Catalogue 中注册并通过校验
