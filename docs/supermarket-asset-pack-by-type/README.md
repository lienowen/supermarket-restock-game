# Supermarket Asset Pack (Type-Based Structure)

This package contains 24 Day2-related supermarket scene assets, organized by **asset type** instead of by day.

- `backgrounds/store/`
- `backgrounds/navigation/`
- `backgrounds/checkout/`
- `stockroom/`
- `restock/drinks/`
- `restock/dairy/`
- `restock/frozen/`
- `restock/snacks/`
- `restock/pantry/`

## Notes
- This structure is **not** split by Day folders.
- It is intended to make project integration easier by grouping assets according to scene or gameplay role.
- Several files are suitable as main backgrounds, while the drinks set already works as an `empty -> mid -> full` progression.
